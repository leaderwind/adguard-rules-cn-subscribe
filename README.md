# AdGuard Rules CN Subscribe

面向中文用户的 AdGuard 公开订阅仓库。规则由私有维护仓库完成上游获取、分层、去重、审核和生成，本仓库只发布最终 `dist/` 与订阅说明。

规则约每 7 小时自动更新。
